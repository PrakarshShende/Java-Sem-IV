## ⚙️ Setup & Usage

### Prerequisites

```
JDK 8 or higher
IDE — IntelliJ IDEA / Eclipse / VS Code (Java Extension Pack)
```

### Compile & Run

```bash
# Compile a single file
javac FileName.java

# Run the compiled class
java FileName

# Compile a file inside a package (run from project root)
javac packageName/FileName.java

# Run a class inside a package
java packageName.FileName
```
