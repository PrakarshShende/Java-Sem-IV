## 01 · Object-Oriented Programming Basics

> Classes, objects, constructors, and getter–setter methods demonstrating core Java encapsulation.

| Program | Description |
|---|---|
| 🏢 **Employee Salary** | Calculate yearly salary and apply a 10% raise using class methods |
| 📐 **Rectangle Area** | Calculate the area of a rectangle using class-based methods |
| 🔒 **Static Members** | Demonstrate static variables, methods, and static blocks |
| 📚 **Stack & Queue** | Implement classic stack and queue data structure operations |
| 🔤 **Name Sorter** | Arrange a list of names in alphabetical order |

**Concepts:** `Encapsulation` `Object Creation` `Constructors` `Getter / Setter`

## ⚙️ Setup

### Prerequisites

- **Java Development Kit (JDK)** — version 8 or higher
- **IDE** — IntelliJ IDEA, Eclipse, or VS Code with the Java Extension Pack

### Running Programs

```bash
# Compile a file
javac FileName.java

# Run the compiled class
java FileName

# Compile with package structure (run from project root)
javac packageName/FileName.java

# Run a class inside a package
java packageName.FileName
```
